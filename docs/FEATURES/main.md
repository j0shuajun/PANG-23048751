# 메인 화면 구성 (초안)

## 목적

사용자가 게임을 실행했을 때 가장 먼저 보는 화면으로, 게임을 시작하기 전 조작법을 안내하고 플레이로 진입시키는 것이 목적이다.

## 화면 구성 요소

- **타이틀**: 게임 이름(PANG) 표시
- **시작 안내 문구**: 예) "Space 키를 눌러 시작"
- **조작법 안내**
  - 이동: 방향키 좌/우
  - 발사: Space 키
  - 목표: 화면의 모든 Bubble을 제거하면 승리
- **레이아웃**: 화면 중앙 정렬, 단순한 단색 또는 배경 이미지 1장 정도의 미니멀한 구성

## 화면 전환

- 시작 입력(Space 키 등)을 받으면 Mission 1 게임 화면으로 전환된다.

## 참고 / 가정

- 최고 점수, 로그인, 설정 메뉴 등은 이번 KATA 범위에 포함하지 않는 것으로 가정한다(교안에서 별도 언급 없음).
- 위 구성은 초안이며, PLAN.md 작성 전 검토를 통해 확정한다.
