# 메인 화면 구성 (초안)

## 목적

사용자가 게임을 실행했을 때 가장 먼저 보는 화면으로, 게임을 시작하기 전 조작법을 안내하고 플레이로 진입시키는 것이 목적이다.

## 화면 구성 요소

- **타이틀**: 게임 이름(PANG) 표시
- **시작 안내 문구**: 예) "Space 키를 눌러 시작"
- **조작법 안내**
  - 이동: 방향키 좌/우
  - 발사: Space 키
  - 목표: 화면의 모든 Bubble을 제거하면 승리
- **레이아웃**: 화면 중앙 정렬, 단순한 단색 또는 배경 이미지 1장 정도의 미니멀한 구성

## 메뉴 탐색 (키보드 방향키)

게임 선택 목록과 시작/종료 버튼은 하나의 메뉴로 취급하며, 마우스 없이 키보드만으로 이동할 수 있어야 한다.

- **메뉴 항목 순서**(위 → 아래): 게임 선택 목록의 각 미션 항목(현재는 Mission 1 하나) → 시작 버튼 → 종료 버튼
- **↓ 키**: 다음 항목으로 포커스 이동. 마지막 항목(종료 버튼)에서 누르면 첫 항목으로 순환 이동한다.
- **↑ 키**: 이전 항목으로 포커스 이동. 첫 항목(첫 번째 미션)에서 누르면 마지막 항목으로 순환 이동한다.
- 현재 포커스된 항목은 다른 항목과 구분되도록 시각적으로 강조 표시된다.
- 포커스된 항목을 Enter/Space 등으로 "실행"하는 동작은 이번 범위에 포함하지 않는다(메뉴 이동만 다룸).

## 화면 전환

- 시작 입력(Space 키 등)을 받으면 Mission 1 게임 화면으로 전환된다.

## 참고 / 가정

- 최고 점수, 로그인, 설정 메뉴 등은 이번 KATA 범위에 포함하지 않는 것으로 가정한다(교안에서 별도 언급 없음).
- 메뉴 탐색은 위/아래 방향키만 사용한다고 가정한다. 좌/우 방향키는 이후 게임 플레이 화면에서 Player 이동으로 사용되므로 메인 화면 메뉴에서는 사용하지 않는다.
- 메뉴 끝에서 반대쪽 끝으로 순환 이동하는 방식은 초안이며, 검토 후 "끝에서 멈춤" 방식으로 바뀔 수 있다.
- 위 구성은 초안이며, PLAN.md 작성 전 검토를 통해 확정한다.
