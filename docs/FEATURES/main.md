# 메인 화면 구성 (초안)

## 목적

사용자가 게임을 실행했을 때 가장 먼저 보는 화면으로, 게임을 시작하기 전 조작법을 안내하고 플레이로 진입시키는 것이 목적이다.

## 화면 구성 요소

- **타이틀**: 게임 이름(PANG) 표시
- **시작 안내 문구**: 예) "Space 키를 눌러 시작"
- **조작법 안내**
  - 이동: 방향키 좌/우
  - 발사: Space 키
  - 목표: 화면의 모든 Bubble을 제거하면 승리
- **레이아웃**: 화면 중앙 정렬, 단순한 단색 또는 배경 이미지 1장 정도의 미니멀한 구성

## 메뉴 탐색 (키보드 방향키)

메인 화면 메뉴는 "게임 선택 / 시작 / 종료" 3개 행으로 구성하며, 마우스 없이 키보드만으로 이동·조작할 수 있어야 한다. Mission이 여러 개(Mission 1~5)로 늘어난 뒤에도 위/아래 이동이 번거롭지 않도록, 미션 전환은 좌/우 방향키로 분리했다.

- **행 순서**(위 → 아래): 게임 선택 → 시작 버튼 → 종료 버튼
- **↓ / ↑ 키**: 세 행 사이를 순환 이동한다(마지막 행에서 ↓ 누르면 첫 행으로, 첫 행에서 ↑ 누르면 마지막 행으로).
- **← / → 키**: "게임 선택" 행에 포커스가 있을 때만 동작하며, Mission 1~5 사이를 순환 전환한다. 다른 행에 포커스가 있을 때는 좌/우 입력이 무시된다.
- 현재 포커스된 행과 현재 선택된 미션은 각각 시각적으로 구분되어 표시된다(게임 선택 행은 "◀ Mission N ▶" 형태로 선택된 미션 이름만 보여준다).
- Enter는 포커스가 "시작" 행이면 선택된 미션으로 게임을 시작하고, "종료" 행이면 종료 동작을 실행한다. "게임 선택" 행에서 Enter는 별도 동작이 없다(미션은 좌/우로 이미 확정됨).

## 화면 전환

- 시작 입력(Space 키 등)을 받으면 Mission 1 게임 화면으로 전환된다.

## 참고 / 가정

- 최고 점수, 로그인, 설정 메뉴 등은 이번 KATA 범위에 포함하지 않는 것으로 가정한다(교안에서 별도 언급 없음).
- 좌/우 방향키는 게임 선택 행에서 미션 전환 용도로만 쓰이며, 게임 플레이 화면(Player 이동)의 좌/우 방향키와는 화면이 다르므로 충돌하지 않는다.
- 메뉴 끝에서 반대쪽 끝으로 순환 이동하는 방식(행 이동, 미션 전환 모두)은 초안이며, 검토 후 "끝에서 멈춤" 방식으로 바뀔 수 있다.
- 위 구성은 초안이며, PLAN.md 작성 전 검토를 통해 확정한다.
