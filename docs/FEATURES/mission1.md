# Mission 1 난이도 및 규칙 (초안)

Mission 1은 플레이어가 처음 만나는 미션으로, `game_rule.md`의 기본 규칙을 그대로 사용하되 **가장 쉬운 난이도**로 구성해 이동·발사·Bubble 분할 규칙을 익히도록 하는 것이 목적이다.

## 난이도 설계 방향

- 기본 규칙(Player/Wire/Bubble/Block)에서 추가/변경되는 규칙은 없다.
- 초반 학습 목적에 맞춰 Bubble 수와 Block 배치를 최소화한다.

## Mission 1 구성 (초안, 확정 아님)

- **시작 Bubble 개수**: 1~2개
- **시작 Bubble 크기**: 가장 큰 크기 1단계에서 시작(분할 여지가 크지 않도록)
- **Block 배치**: 없음 또는 화면 하단에 최소한의 발판 1개 수준
- **클리어 조건**: 화면의 모든 Bubble(분할된 것 포함)을 제거하면 클리어

## 참고 / 가정

- 위 수치(Bubble 개수/크기, Block 배치)는 원본 게임이나 교안에 명시된 값이 아니라, "첫 미션은 가장 쉬워야 한다"는 목적에 맞춰 임시로 설정한 초안이다.
- 실제 값은 PLAN.md 작성 및 사람 검토 단계에서 확정한다.
