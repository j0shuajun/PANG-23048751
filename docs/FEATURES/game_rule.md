# 게임 룰 상세 내용 (초안)

`docs/PRD.md`에 정리된 Pang KATA 규칙을 기반으로, 구현을 염두에 두고 조금 더 상세하게 정리한다.

## Object별 상세 규칙

### Player
- 방향키 좌/우로만 이동하며, 위/아래로는 이동하지 않는다.
- Space 키를 누르면 현재 위치에서 수직 위 방향으로 Wire를 발사한다.
- Bubble에 몸이 닿으면 목숨을 1개 잃는다.
- 목숨이 모두 소진되면 게임 오버로 처리한다. *(가정: 시작 목숨 수는 3개로 임시 설정, 확정값 아님)*

### Bubble
- 좌/우로 튕기며 이동하고, 중력의 영향을 받아 포물선을 그리며 낙하와 반등을 반복한다.
- 튕길 때마다 직전과 같은 높이까지 올라간다(에너지 손실 없음).
- 크기는 단계적으로 나뉜다: 큰 Bubble → Wire에 맞으면 중간 크기 2개로 분할 → 다시 Wire에 맞으면 더 작은 크기 2개로 분할 → 가장 작은 크기는 Wire에 맞으면 소멸. *(가정: 분할 단계 수는 원본과 동일하게 최대 2단계 분할로 임시 설정, 세부 단계 수는 PLAN.md 단계에서 확정)*
- 화면 좌/우/상단 경계에서는 반사되고, 화면 밖으로 나가지 않는다.
- Block 위쪽 면에서는 튕겨 올라갈 수 있다.

### Wire
- Space 키 입력 시 Player 위치에서 수직 위 방향으로 즉시 발사된다.
- Block에 닿으면 그 지점에서 소멸한다(벽을 뚫지 못함).
- Bubble에 닿으면 해당 Bubble을 규칙에 따라 분할시키거나 소멸시키고, Wire 자신도 함께 사라진다.

### Block
- 파괴되지 않는 고정 장애물이다.
- Bubble의 반등면으로 사용되며, Wire의 사거리를 막는 역할을 한다.

## 승리 / 패배 조건

- **승리**: 화면에 남아있는 모든 Bubble을 제거하면 해당 미션을 클리어한다.
- **패배**: 목숨이 모두 소진되면 게임 오버 화면으로 전환된다.

## 참고 / 가정

- 점수(Score) 체계, 제한 시간 유무는 교안에 명시되어 있지 않아 이번 초안 범위에서는 제외한다.
- 목숨 수, 분할 단계 수 등 구체적인 수치는 임시 가정이며 PLAN.md 작성 시 사람 검토를 통해 확정한다.
