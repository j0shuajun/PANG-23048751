# PANG (팡) 게임 PRD

## 1. 개요

- **장르**: 아케이드 액션 + 퍼즐 (반사신경과 판단력을 함께 요구)
- **한 줄 요약**: 사파리 복장의 캐릭터가 화면 위로 작살을 쏘아 통통 튀는 풍선을 모두 터뜨리면 스테이지를 클리어하는 게임

## 2. 핵심 게임 규칙

- 캐릭터는 좌우로만 이동하며, 버튼을 누르면 위쪽으로 작살(와이어)을 발사한다.
- 화면 안에는 중력의 영향을 받아 통통 튀는 풍선(공)이 떠다닌다.
- 풍선을 작살로 맞히면 풍선이 2개의 더 작은 풍선으로 분열되며, 이 과정을 최대 3번까지 반복한 뒤 가장 작은 풍선을 맞히면 완전히 사라진다.
- 캐릭터 몸에 풍선이 닿으면 목숨을 하나 잃는다.
- 화면에 있는 풍선을 전부 없애면 스테이지 클리어, 다음 스테이지로 넘어간다.

## 3. 스테이지 구성과 Mission 1

- 전 세계 유명 장소를 배경으로 총 17개 지역, 지역당 3스테이지씩 이어지는 구조(총 50스테이지)로 구성되어 있다.
- **Mission 1(첫 지역)의 배경은 일본 후지산**이며, 이후 타지마할, 파리, 런던, 이스터섬 등 세계 각지 랜드마크로 이어진다.
- 초반 스테이지는 풍선 개수가 적고 장애물이 단순해 조작(이동 + 발사)과 분열 규칙을 익히는 난이도로 설계되어 있으며, 지역이 바뀔수록 풍선 수와 장애물 배치가 복잡해진다.
- 스테이지마다 파괴 가능한 블록과 파괴되지 않는 고정 블록이 섞여 있어, 이동 동선과 작살 사거리를 함께 고려해야 한다.

## 4. 아이템 / 파워업

| 종류 | 효과 |
| --- | --- |
| Double Wire | 작살 두 줄기를 동시에 발사 |
| Power Wire | 잠깐 벽/천장에 걸리는 그래플링 형태의 작살 |
| Vulcan Missile | 연속 발사가 가능한 기관총형 무기로 일시 전환 |
| Forcefield | 일정 시간 풍선 접촉으로부터 보호막 제공 |
| 시간 정지(정지 시계) | 풍선의 움직임을 잠시 멈춤 |
| 다이너마이트 | 화면 내 모든 풍선을 즉시 가장 작은 크기로 축소 |
| 과일 등 보너스 아이템 | 점수 획득용 |
| 추가 목숨 | 잔여 목숨 1개 증가 |
